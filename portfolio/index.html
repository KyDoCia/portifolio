Enhanced Player Movement

PlayerMovement.lua 
local player = game.Players.LocalPlayer
local character = player.Character or player.CharacterAdded:Wait()
local humanoid = character:FindFirstChild('Humanoid')

humanoid.WalkSpeed = 50
humanoid.JumpPower = 100

print('Player movement enhanced!')

This script enhances player movement by increasing walk speed and jump power, providing a more dynamic gaming experience.

Data Persistence with DataStore

DataStore.lua 
local DataStoreService = game:GetService('DataStoreService')
local playerDataStore = DataStoreService:GetDataStore('PlayerData')

local function savePlayerData(player, data)
    local success, errorMessage = pcall(function()
        playerDataStore:SetAsync(player.UserId, data)
    end)
    if not success then
        warn('Failed to save data: ' .. errorMessage)
    end
end

local function loadPlayerData(player)
    local success, data = pcall(function()
        return playerDataStore:GetAsync(player.UserId)
    end)
    if success then
        return data or {}
    else
        warn('Failed to load data')
        return {}
    end
end

This script shows how to save and load player data using Roblox's DataStore service, essential for persistent game progress.

Weapon System Implementation

WeaponSystem.lua 
local tool = script.Parent

tool.Activated:Connect(function()
    local player = tool.Parent.Parent
    local character = player.Character
    local humanoidRootPart = character:FindFirstChild('HumanoidRootPart')

    -- Create a projectile
    local projectile = Instance.new('Part')
    projectile.Size = Vector3.new(0.5, 0.5, 2)
    projectile.Position = humanoidRootPart.Position + humanoidRootPart.CFrame.LookVector * 2
    projectile.Anchored = false
    projectile.CanCollide = true
    projectile.Parent = workspace

    -- Add velocity to the projectile
    local bodyVelocity = Instance.new('BodyVelocity')
    bodyVelocity.Velocity = humanoidRootPart.CFrame.LookVector * 100
    bodyVelocity.MaxForce = Vector3.new(4000, 4000, 4000)
    bodyVelocity.Parent = projectile

    -- Remove projectile after 5 seconds
    wait(5)
    projectile:Destroy()
end)

This weapon system script creates a projectile when the tool is activated, demonstrating basic combat mechanics in Roblox games.
